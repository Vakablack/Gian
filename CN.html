ECE 3rd Year Class

Syllabus

  • Electronic Circuits and Systems
  • Digital Signal Processing
  • Communication Systems
  • Control Systems
  • Microprocessors and Microcontrollers
  • Electromagnetic Theory
  • Antennas and Wave Propagation

Assignments

Electronic Circuits and Systems

  • Assignment 1: Design and simulate a single-stage common-emitter amplifier using a BJT transistor
  • Assignment 2: Design and simulate an op-amp based summing amplifier circuit
  • Assignment 3: Design and simulate a low-pass filter using an active op-amp circuit

Digital Signal Processing

  • Assignment 1: Implement a digital FIR filter in MATLAB for noise reduction in an audio signal
  • Assignment 2: Implement a digital IIR filter in MATLAB for speech signal processing
  • Assignment 3: Design and simulate a discrete Fourier transform (DFT) algorithm in MATLAB

Communication Systems

  • Assignment 1: Simulate the performance of different modulation schemes in AWGN channel using MATLAB
  • Assignment 2: Design and simulate a phase-locked loop (PLL) for carrier synchronization in a communication system
  • Assignment 3: Implement a basic digital communication system using MATLAB and Simulink

Control Systems

  • Assignment 1: Design and simulate a PID controller for a DC motor system using MATLAB and Simulink
  • Assignment 2: Analyze the stability of a control system using Routh-Hurwitz criterion
  • Assignment 3: Design and simulate a state feedback controller for a linear time-invariant (LTI) system

Microprocessors and Microcontrollers

  • Assignment 1: Write a program to control the operation of an LED using 8051 microcontroller
  • Assignment 2: Interface a 7-segment display with the ARM Cortex-M3 microcontroller
  • Assignment 3: Design and implement a real-time embedded system using a microcontroller

Electromagnetic Theory

  • Assignment 1: Solve problems related to electromagnetic waves and their propagation in different media
  • Assignment 2: Analyze the behavior of transmission lines and waveguides using Maxwell's equations
  • Assignment 3: Design and simulate an antenna array for directional beamforming

Antennas and Wave Propagation

  • Assignment 1: Design and simulate a microstrip patch antenna using HFSS software
  • Assignment 2: Investigate the propagation of radio waves in different environments using MATLAB
  • Assignment 3: Design and simulate a satellite communication system using antenna arrays and signal processing techniques
